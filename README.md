## 🕵️ Cybersecurity mindset

Security is not just about tools. Tools are easy to run. Understanding the system, the assumptions, and the failure points is the actual skill.

I try to approach problems with this mindset:

```text
Question assumptions.
Read the error messages.
Verify before trusting.
Understand before automating.
Document what confused you.
```
